Comment trouver l'ID d'un Item ?

Le moyen le plus simple pour copier n'importe quel ID (même des mods) :

/ct hand
  1. Tiens l'objet dans ta main dans Minecraft.
  2. Tape /ct hand dans le chat.
  3. Dans le message qui apparaît, clique sur le texte "<item:...>" (ou le nom) pour copier l'ID.
  4. Colle-le ici dans le champ "Object Name" !
